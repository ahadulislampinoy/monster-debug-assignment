The event loop is the secret behind JavaScript‚Äôs asynchronous programming. JS executes all operations on a single thread, but using a few smart data structures, it gives us the illusion of multi-threading. Whenever an async function is called, it is sent to a browser API. Based on the command received from the call stack, the API starts its own single-threaded operation. It constantly checks whether or not the call stack is empty. If it is empty, new functions are added from the event queue. If it is not, then the current function call is processed.
